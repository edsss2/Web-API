# Sistema de Movimentações (Back-end)

![Status](https://img.shields.io/badge/Status-Concluído-brightgreen)
![Node.js](https://img.shields.io/badge/Node.js-339933?logo=nodedotjs&logoColor=white)
![MongoDB](https://img.shields.io/badge/MongoDB-4ea94b?logo=mongodb&logoColor=white)

Back-end desenvolvido em **Node.js** com **MongoDB** para um CRUD simples de movimentações financeiras.  
Responsável por cadastrar, listar e excluir movimentações, além de calcular o saldo total.

> **Nota:** Este projeto foi desenvolvido para a prova final da disciplina de Desenvolvimento Web Front-end da faculdade.

Front-end do projeto: [link para o repositório](https://github.com/edsss2/AF-WEB.git)
